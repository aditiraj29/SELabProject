# 🔬 EVIDENTIA — Digital Evidence Tampering Detection System
[Live Demo](https://se-lab-project-flax.vercel.app/)
 
EVIDENTIA is a client-side forensic tool designed to detect potential tampering in digital evidence using cryptographic hashing and heuristic analysis techniques.

## 🚀 Features

- 📂 File Upload & Metadata Analysis
- 🔐 Cryptographic Hashing (MD5, SHA-256, SHA-512)
- 🧪 Forensic Analysis Engine (8 Modules)
- ⚠️ Tampering Detection & Threat Scoring
- 🔍 Hash Comparison Tool
- 📊 Pixel Anomaly Heatmap (Simulated)
- 📝 Automated Forensic Case Report Generator
- ⚡ Live Hash Demo (Avalanche Effect)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- CryptoJS (for hashing)

## 🌐 How It Works

1. Upload a file
2. System generates hash values
3. Runs forensic checks
4. Displays tampering verdict
5. Generates a detailed forensic report

## 🔒 Privacy

- 100% client-side processing
- No data leaves the browser

## 📦 Deployment

Deployed using Vercel via GitHub integration.
